== Description ==

= Sitewide / Network-wide Galleries and pictures =

This plugin replicates all NGG galleries and images to your SWT aggregation blog.

It is an add-on to the [NextGEN gallery plugin](http://wordpress.org/extend/plugins/nextgen-gallery/) by [alexrabe](http://profiles.wordpress.org/users/alexrabe/). 
It will not work and is not useful if you do not have the NextGEN plugin installed.

Compatible with PHP5.

== Installation ==

Wordpress automatic installation is fully supported and recommended.
